Online image color picker (RGBA)


Current mode: Move mode

Move mode: Update the RGBA values whenever the cursor move on the image
Click mode: Only update the RGBA values after left click on the image

The submitted image should only be processed in your device, I will not be able to access it, feel free to check the source code at https://github.com/ChanYatChunSunny/ChanYatChunSunny.github.io if you are concerned.
